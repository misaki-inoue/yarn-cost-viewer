# Yarn Cost Viewer
## Project Summary
This React web application is used to calculate the cost (yen/meter) of different yarns listed on Rakuten. I decided to make this app because I wanted to easily find the cheapest yarn for its length of yarn, not per skein. This app will use Rakuten API to obtain a list of yarn that match the search query and their product description to calculate the meterage cost.

### Pain of using Rakuten
When you search for a type of yarn in Rakuten, it is not very obvious how big the skein is. Also, different shops can carry the same yarn product with different yarn cost and shipping cost. Reordering the search result by lowest to highest cost does not guarantee that you are getting the best deal.

## Tech Stack
This project is built using Reect and TypeScript. The web app is hosted on Amplify.
